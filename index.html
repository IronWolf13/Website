Règlement du Serveur

Règles de base

  • Il est interdit de tuer gratuitement (tuer sans raison un autre joueur, les instances de SCP‑049 ne sont pas considérées comme freekill si elles se font tuer pour rien).
  • Il est interdit de faire du méta‑gaming (connaître une information que tu n'es pas censé savoir).
  • Il est interdit de camper en surface pour attendre les F.I.M, U.I.U, Chaos ou Main du serpent.
  • Il est interdit de tirer gratuitement sur un allié.
  • Il est interdit de ne pas simuler la douleur (subir des dégâts sans exprimer de douleur, sauf pour certaines classes militaires qui doivent jouer la douleur à 15 PV ou moins).
  • Il est obligatoire de respecter la peur des armes, sauf si vous en tenez une (exception pour certaines classes).
  • Il est interdit de troller (ex. : fermer les portes, tirer n'importe où, hurler pour rien, etc.).
  • Durant le RP, un comportement sérieux est impératif (le HRP vocal est compris).
  • Il est interdit de bunny hop (effectuer 3 sauts ou plus trop rapidement).
  • Il est interdit d'exploiter un bug (ex. : ouvrir le confinement de SCP‑079 alors que c’est confiné).
  • Il est obligatoire de faire trois sommations avant de tuer une personne (sauf exceptions).
  • Il est interdit de tenir des propos déplacés (ex. : vocabulaire inapproprié, insultes, etc.).
  • Il est interdit de courir si vous avez 15 PV ou moins (sauf cas particuliers).
  • Il est interdit de s'évacuer en Chaos ou F.I.M au début du respawn pour ne pas jouer son rôle (sauf sur ordre).
  • Il est interdit de déco‑reco pour éviter de jouer son rôle.
  • En cas d'infraction, faites un ticket et n'intervenez pas sans l'autorisation d'un staff.
  • Il est interdit de menotter un membre de son équipe sans raison valable.
  • Il est interdit de dégainer son arme dans certaines situations (face à un mur et lorsqu'une personne vous pointe l'arme).
  • Les logiciels de triche sont bannissables à vie.
  • Les parties sont RP à partir de 7 personnes.
  • Il est interdit de menotter les Classe‑D sans raison valable.
  • Il est interdit d'utiliser des soundboard ou voice mod.
  • Voler ses alliés est interdit (sauf pour les Classe‑D).
  • Il est interdit de tuer un allié sans raison valable.
  • Il est interdit de tuer le preneur d'otage sauf dans certaines conditions.
  • Il est interdit de faire du power‑gaming (actions impossibles en vrai).
  • Il est interdit de gueuler dans son micro pour rien.
  • Les membres de la fondation ne doivent pas évacuer sans brèche (sauf exceptions).

Protocoles

  • En cas de déconfinement majeur (4 SCP minimum et hostiles), le protocole de l'ogive alpha doit être déclenché.
  • En cas d’intrusion sur site, tous les objets SCP doivent être évacués.
  • Si un SCP est en surface, les membres militaires de la fondation doivent l’abattre (sauf exception si le SCP peut communiquer).
  • En cas de code rouge (brèche de confinement), le personnel non combattant doit évacuer.

Commandes

Fonctionnalité/commande :

  • Si vous êtes un Insurgé du chaos, faites .hack dans la salle des serveurs pour les pirater et déconfiner SCP‑079.
  • Si vous êtes Insurgé du chaos, faites .copy pour copier SCP‑079 et l'évacuer en sortant de la section souterraine du site.
  • Si vous êtes Capitaine F.I.M ou Commandant UIU, faites .decontamination dans la salle des serveurs pour activer la décontamination.
  • Si vous êtes Insurgé du chaos, Main du serpent ou Garde infiltré, faites .106 pour pirater le confinement de SCP‑106 et le libérer.
  • Si vous êtes un humain, faites .sacrifice pour vous sacrifier dans le femur breaker.
  • Si vous êtes un humain, faites .activate pour activer le femur breaker.
  • Si vous êtes un concierge et chef DI&ST :
    • Faites .clean pour nettoyer le corps le plus proche.
    • Faites .iclean pour supprimer l'objet que vous visez.
  • Si vous êtes un humain :
    • Faites .294 [nom de la boisson] (ex. : .294 water) avec une pièce dans la main devant le SCP pour obtenir votre boisson (plus de 100 commandes disponibles).
    • Faites .punch pour frapper de façon RP les autres joueurs (respectez le FearRP).
    • Faites .push pour pousser de façon RP les autres joueurs.
    • Faites .steal pour tenter de voler un objet dans l'inventaire d'un autre joueur (à vos risques).
    • Faites .cuff avec une arme pour menotter un joueur de votre équipe.
    • Faites .heal sur un SCP avec un objet destiné à soigner.
  • Si vous êtes SCP‑079 :
    • Faites .simulate [ci/mtf/recont] pour envoyer un faux message via C.A.S.S.I.E.
    • Faites .find [id joueur/nom joueur] pour vous téléporter dans la même salle que la cible.
    • Faites .overload pour désactiver les générateurs en activation.
    • Faites .overcharge [1-60] pour éteindre les lumières du site pendant un temps donné (max 60 sec).
    • Faites .gaz pour activer le système de gaz dans une salle.
    • Faites .flash pour flasher avec la caméra les personnes qui vous regardent.
  • Si vous êtes un SCP :
    • Faites .conf pour vous reconfiner en SCP.
  • Si vous êtes SCP‑049, appuyez sur [ALT] pour activer le chat de proximité et parler aux humains.
  • Si vous êtes SCP‑999, appuyez sur [ALT] pour activer votre capacité de soins.
  • Si vous êtes SCP‑575, appuyez sur [ALT] pour activer votre capacité et éteindre toutes les lumières.

Révocations

Cette page est dédiée aux demandes de révocation de sanction (warn, ban temporaire ou ban permanent).
Pour toute demande, veuillez remplir le questionnaire suivant : https://form.jotform.com/240366450124347.
Si vous êtes sanctionné(e) de façon injuste, votre sanction pourra être retirée. Sinon, ne tentez pas de demander une révocation.
